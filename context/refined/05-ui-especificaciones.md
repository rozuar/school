# Especificaciones de UI

## Listado de Alumnos
- Usar icono de botones en expansion
- Al expandir aparecen los botones con los comportamientos disponibles

## Sistema de Semaforo Visual (iconos de personas)

| Estado | Color | Icono |
|--------|-------|-------|
| Ausente | Negro | Persona pequena |
| Bano / Enfermeria | Amarillo | Persona pequena |
| SOS / Comportamiento grave | Rojo | Persona pequena |

**Nota:** Los iconos deben ser pequenas personas, no puntos.

## Dashboards
- Estados visuales con colores e iconos
- UI orientada a rapidez operativa
