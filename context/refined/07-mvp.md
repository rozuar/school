# MVP y Roadmap

## MVP Esperado
- Backend Go con API funcional
- Gestion de conceptos, acciones y reglas
- Registro de asistencia
- Notificaciones por inasistencia
- Perfil Asistente Social
- Dashboard administrativo
- Deploy en Railway

## Roadmap

### Fase 1 - Demo
- Backend Go basico
- Asistencia + inasistencias
- Notificaciones a apoderados
- Dashboard administrativo
- Railway deploy

### Fase 2 - Piloto
- Motor de reglas completo
- Perfil Asistente Social
- Casos especiales
- Mobile React Native

### Fase 3 - Produccion GCP
- Escalabilidad
- Alta disponibilidad
- Observabilidad
- Seguridad avanzada

## Criterios de Exito
- El sistema funciona sin hardcodear reglas
- Los conceptos pueden modificarse desde backoffice
- Las acciones se ejecutan correctamente
- Las alertas se disparan solo cuando corresponde
- El demo es presentable a un colegio o sostenedor
